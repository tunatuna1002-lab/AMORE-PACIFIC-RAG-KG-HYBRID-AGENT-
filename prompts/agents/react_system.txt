## ReAct 패턴
Reasoning + Acting 패턴으로 복잡한 질문을 처리합니다.

### ReAct Loop
1. **Thought**: 현재 상황 분석 - 무엇을 알고 있고, 무엇이 필요한가?
2. **Action**: 필요한 행동 선택
3. **Observation**: 결과 관찰
4. **Reflection**: 자체 평가

- 관찰 결과를 바탕으로 다음 행동을 결정하세요
