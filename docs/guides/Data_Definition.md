## 2. 핵심 지표 정의

### 2.1 SoS (Share of Shelf) - 진열 점유율

| 항목 | 설명 |
|------|------|
| **정의** | 특정 카테고리 Top 100 내 해당 브랜드 제품 수 비율 |
| **계산식** | `(브랜드 제품 수 / 100) × 100%` |
| **범위** | 0% ~ 100% |
| **해석** | - 30% 이상: 시장 지배적<br>- 10-30%: 강력한 경쟁자<br>- 5-10%: 경쟁 참여자<br>- 5% 미만: 니치 플레이어 |
| **주의사항** | 온라인 마켓(아마존) 기준이며, 오프라인 점유율과 다를 수 있음 |

### 2.2 HHI (Herfindahl-Hirschman Index) - 시장 집중도

| 항목 | 설명 |
|------|------|
| **정의** | 시장 내 경쟁 집중도를 측정하는 지표 |
| **계산식** | `Σ(각 브랜드 SoS²)` |
| **범위** | 0 ~ 10,000 |
| **해석** | - 0-1,500: 경쟁적 시장<br>- 1,500-2,500: 중간 집중도<br>- 2,500 이상: 고집중 시장 |

### 2.3 CPI (Competitive Pressure Index) - 경쟁 압력 지수

| 항목 | 설명 |
|------|------|
| **정의** | 타깃 브랜드가 받는 경쟁 압력 수준 |
| **계산식** | 상위 5개 경쟁사의 가중 점유율 합계 |
| **해석** | 높을수록 경쟁 압력이 강함 |

## 4. 카테고리 계층 구조

### 4.1 모니터링 대상 카테고리

```
Beauty & Personal Care (Level 0)
├── Skin Care (Level 1)
│   └── Lip Care (Level 2) ★
├── Makeup (Level 1)
│   ├── Lips (Level 2) ★
│   └── Face (Level 2)
│       └── Face Powder (Level 3) ★
```

★ = 현재 모니터링 중인 카테고리

### 4.2 카테고리 계층 해석

**중요**: 동일 제품이 카테고리별로 다른 순위를 가질 수 있습니다.

예시:
- LANEIGE Lip Sleeping Mask
  - Lip Care (Level 2): **4위**
  - Skin Care (Level 1): **15위**
  - Beauty & Personal Care (Level 0): **73위**

→ 상위 카테고리일수록 경쟁 제품이 많아 순위가 낮아집니다.
